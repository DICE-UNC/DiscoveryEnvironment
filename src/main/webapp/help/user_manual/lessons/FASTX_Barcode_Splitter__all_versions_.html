FASTX Barcode Splitter (all versions)

An overview of FASTX Analyses is available.

The FASTX Barcode Splitter splits a FASTQ file into several files using barcodes as the split criteria.

Barcode files are simple text files. Each line should contain an identifier (descriptive name for the barcode), and the barcode itself (A/C/G/T), separated by a TAB character." An example is given in an image on the FASTX documentation website.

Select the version of the FASTX Barcode Splitter that is appropriate for your input file(s). Use:
• Single End if you have single end reads
• Concatenated if you have paired end reads already concatenated into one file (not yet available)
• Paired End with mate file if you have paired end reads in two separate files (not yet available)

Click Ok.
