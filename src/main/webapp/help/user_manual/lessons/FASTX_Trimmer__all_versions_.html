FASTX Trimmer (all versions)

An overview of FASTX Analyses is available.

Select the version of the FASTX Barcode Trimmer that is appropriate for your input file(s). Use:
• Single End if you have single end reads
• Concatenated if you have paired end reads already concatenated into one file (not yet available)
• Paired End with mate file if you have paired end reads in two separate files (not yet available)

Click Ok.

Single end read data input

Zoom

Click Browse to select your previously uploaded file. Click Remove Non-Biological Sequences.

Concatenated paired end read data input

Click Browse to select your previously uploaded file. Click Remove Non-Biological Sequences.

Paired end with mate file data input

Click Browse to select your previously uploaded file and its mate pair. Click Remove Non-Biological Sequences.

Zoom

Keep or modify the default settings. Click Launch Job.

Enter a name and desciription for the job and click OK. See Perform Analyses for information about monitoring the process and where to find your results.
