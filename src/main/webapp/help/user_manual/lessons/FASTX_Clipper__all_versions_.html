FASTX Clipper (all versions)

An overview of FASTX Analyses is available.

Select the version of the FASTX Clipper that is appropriate for your input file(s). Use:
• Single End if you have single end reads
• Concatenated if you have paired end reads already concatenated into one file (not yet available)
• Paired End with mate file if you have paired end reads in two separate files (not yet available)

Click Ok.

Single end read data input

Zoom

Click Browse to select your previously uploaded file. Click Trim 3' Adapters.

Concatenated paired end read data input

Click Browse to select your previously uploaded file. Click Trim 3' Adapters.

Paired end with mate file data input

Click Browse to select your previously uploaded file and its mate pair. Click Trim 3' Adapters.

Zoom

Choose Create 3' Adapter File from the dropdown menu if you are going to create one now.

Zoom

Choose Select 3' Adapter File from the drop-down menu if you are going to use a previously uploaded file.

Keep or modify the default settings. Choose your desired output option from the Output options drop-down menu. Click Launch Job.

Enter a name and desciription for the job and click OK. See Perform Analyses for information about monitoring the process and where to find your results.
