Manage Data

Introduction

Zoom

Click the Manage Data icon to upload and manipulate data files.

The window displays all files that you have uploaded or imported into the Discovery Environment, as well as some sample data provided to you by iPlant in the TestData folder.

Create a folder

New Folder creates a new folder.

Import data

Import provides a drop-down menu from which you can upload data from your computer, import data from a URL, or import data from external repositories that have been enabled for direct access from the Discovery Environment, such as the Phylota database provided by the Sanderson lab at the University of Arizona.

Enter upload details

Choose the file from your computer to upload. Select the appropriate filetype from the drop-down list. Click Upload.

File name restrictions:
File names must be unique and may be a maximum of 250 characters. All alphanumeric characters are permitted, along with these special characters: the dash (-), underscore (_) or period (.). Spaces are allowed, but are not permitted as the first, last or only character.

Confirm success

Zoom

Your file will appear in the list of available files.

More actions

Zoom

Mark the check box to the left of an item in this window to expose the drop-down menu shown at the right. Choose the appropriate entry to rename or delete files and folders, view their contents, or download.

If you select one or more check boxes, you may use the More Actions box to perform the same tasks on single files or folders or some actions on multiple files or folders at the same time.

View specific data

Zoom

Depending on the file selected, different tabs will appear in the new window.

For example, viewing a TNRS results file shows a list of names and matches with links. Viewing a .nex file will show Raw and Tree tabs. Viewing a .sam file will show Preview and Description tabs. Other file types display their contents in appropriate ways.
