Bowtie Single End for Illumina (all params)

This analysis uses Bowtie. The configuration options are set to be optimal for reads derived from Illumina sequencing technology, not 454, ABI, or PacBio. It's single-end because the parameterization for paired-end is sufficiently different that an explicit second tool is needed to handle that case.

Select Bowtie Single End for Illumina (all params) from within Perform Analyses as described in that section. Click Ok.
