Burrows-Wheeler Aligner (analysis)

This analysis uses the Burrows-Wheeler Aligner.

Select Burrows-Wheeler Aligner from within Perform Analyses as described in that section. Click Ok.

Select reads

Zoom

Click Browse next to Select Reads to select the previously uploaded and preprocessed DNA sequence read file that you want to align to a reference genome.

Select reference genome

Click the arrow to open a drop-down box listing available reference genomes. Click one to select it.

Click Launch Job.

Enter a name and desciription for the job and click OK. See Perform Analyses for information about monitoring the process and where to find your results.
