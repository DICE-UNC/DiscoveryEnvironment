Independent Contrasts

An overview of Independent Contrasts is available.

Select Independent Contrasts from within Perform Analyses as described in that section. Click Ok.

Select input data

Zoom

Click Add in each section to choose previously uploaded tree and trait files from the boxes shown next. You may use multiple tree files, but only one trait file.

Select Tree or Trees

Highlight your desired file(s) and click Ok.

Select Traits

Highlight your desired file and click Ok.

Match Data

Zoom

All tree data needs to have matching trait data.

Scroll down to drag and drop names as needed to match. When all tree data species is matched with trait data, click Select output details.

Select Output Details

Zoom

Next, click Select Output Details and select Display Correlations and Regressions, Display Contrasts or both.

Select at least one of the options before clicking Launch Job. If neither option is selected, the analysis will still run, however no data will be displayed and the results will not be accessible.

Click Launch Job. See Perform Analyses for information about monitoring the process and where to find your results.
