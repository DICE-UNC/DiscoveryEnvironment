Discrete Ancestral Character Estimation

An overview of Ancestral Character Estimation is available.

Select Discrete Ancestral Character Estimation (DACE) from within Perform Analyses as described in that section. Click Ok.

Select data

Zoom

Data needs to be uploaded to the Discovery Environment in advance.

Click Add to choose appropriate tree and trait files from the boxes shown next.

Select Tree or Trees

Select Traits

Match tree and trait data

Zoom

Drag and drop species trait or tree data up and down until all tree species and trait species are matched. Click Set parameters.

Set parameters

Zoom

You may change the initial value for ML estimation or leave the default value in place.

Click Launch Job.

Enter a name and desciription for the job and click OK. See Perform Analyses for information about monitoring the process and where to find your results.
